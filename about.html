August 4th, 2020

I finished the CSS intro unit today! As I'm sure you've noticed, I added in borders and text color. I even changed the font of this post!

Most of the CSS skills were how to manipulate the html I have on the page already. Most of it was very easy to digest as it's simply telling my html what color to be or what font to be. Border were easy to create but I found them a little more difficult to manipulate. In order to "place" your borders, you have to adjust the padding and margin on them. The padding is what's between the text inside of the border and the border. The margin is the space between the edge of the page and the border. You can either use flat measurements to move it or you can use percentages. I'm honestly not totally sure how either one works, but I found percentage to be easier. I found it easier to visualize my border being 2% away from the edge of the page and my text being 2% away from my border.

I was also able to change the size and shape of my photos. Now, you don't need to see a giant photo of me and Henry. It's smaller and cuter (in my opinion).

Most of the other stuff I learned pertained to different ways to do the same thing. For example, there are many ways to assign a color. You can type a basic color name, you can use the RGB spectrum or you can use Hex Code. None of this made a ton of sense to me so I'm going to stick to the basics for now. There are also multiple ways to priortize different styles and override styles. Again, a little confusing as to why I need all these ways but I assume I'll learn why later down the road (and wish I had paid more attention).

My final note is more of a note to myself than to you. There is an element called a "root" element. I believe is works as a sort of overarching element that controls the document. For example, if I were to put a variable within that root, it would take effect everyone the variable is menitoned. I'm not totally sure of the purpose yet but I do think it's important.

August 22nd,2020

Today has been all about color and how to use it! I've learned another new way to set it (rgba) and I've learned how to set a color based off of hue, lightness and saturation (hsl) which makes it easier to use the same base color with it's different shades. I imagine this will be very helpful in designing block units on websites. Additionally, I've learned how to use a linear gradient(degree,color,color) for instances when I want to show off a little with color. You can make stripes by using the repeating-linear-gradient(angle,color 0px,color50px,color40px,color80px) to set the colors and width of the stripes. Another interesting way to add visual appeal to a webpage is by changing the background. Suprisingly enough, it does not need to be solid. You can use an image url to set the background pattern background: url("").

To change the size of an image, you can use transform(scale) and adjust it in relation to it's original size. You can also change the size of an image in it's hover state to draw more attention as they're hovering over it. Simply add the trnasform:scale property to the hover class of the element. If I wanted an element to be slanted on a page (maybe an italisized paragraph...or a side comment) I can use the transform:skewX/Y function to do that.

Through internent searching, I finally figured out how to change the background color outside of my border! You can use the outline property to set an outline property and then simply make the width of that property large enough to cover your empty space. I'm very excited to now have an additional color.
